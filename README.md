# Postawienie i Uruchomienie Aplikacji

## Wymagania
Do uruchomiania aplikacji należy mieć zainstalowany na systemie `docker` z `docker compose`.

## Postawienie aplikacji

Do zkompilowania i uruchomienia aplikacji należy wykonać komendę:
```
docker compose up --build
```
w głównym katalogu repozytorium.
Aplikacja powinna zostać uruchomiona na adresie `http://localhost:8080/`
